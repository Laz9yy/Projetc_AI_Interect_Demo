## 一、项目概述

### 1.1 核心变更摘要 (v1 → v2)

| 维度 | v1 (当前) | v2 (目标) |
|------|-----------|-----------|
| 脸型 | 宽圆大饼脸 | 纤细鹅蛋幼态脸 |
| 五官比例 | 等分分布 | 低重心 (眼位低、嘴小) |
| 眼睛 | 普通椭圆豆豆眼 | 下垂狗狗眼 + 眼睑层次 |
| 嘴唇 | 宽大扁平微笑嘴 | 小巧柔和浅粉唇 |
| 腮红 | 大面积圆形 | 眼下细竖纹 + 淡色 |
| 发型 | 棕色长发 (4种) | 淡紫锯齿齐刘海短波波头 + 蝴蝶结 |
| 服饰 | 纯色上衣 | 黄短袖内搭 + 淡紫背带套装 |
| 画风 | 粗纯色块 | 细腻渐变 + 浅轮廓阴影 |
| 图层 | 半合并 | 全部件独立分层 (38层) |
| 身体 | 仅半身 | 完整上半身 (头+肩+臂+手+腰) |
| 动态 | 表情切换 | 全表情 + 肢体联动 + 物理待机 |

### 1.2 画风规范

**日系扁平儿童动画平涂风**，核心特征：
- 柔和无硬边描线，使用浅色轮廓阴影替代
- 低饱和马卡龙配色 (紫/粉/浅黄/白)
- 细腻色彩渐变过渡
- 圆润柔和的几何造型

### 3.2 低重心五官比例

```
    纵向比例线
    ┌──────────┐
    │ 0% 额头  │
    │          │
    │ 35% 眉毛 │  ← 眉毛位置偏低
    │          │
    │ 45% 眼睛 │  ← 眼睛在中线偏下 (幼态关键)
    │          │
    │ 58% 鼻头 │
    │          │
    │ 68% 嘴巴 │  ← 嘴靠近下巴 (幼态关键)
    │          │
    │ 78% 下巴 │
    └──────────┘

  关键比例参数:
  眉眼间距 : 面高 = 1 : 8  (宽)
  眼位高度 : 面高 = 45%   (低)
  嘴位高度 : 面高 = 68%   (低)
  嘴宽     : 面宽 = 25%   (小)
```

### 3.3 下垂狗狗眼

**结构分解** (每只眼 4 个独立子层)：

```
┌─────────────────────────────┐
│ ① 上眼睑 + 睫毛 (深紫棕)   │  ← 控制睁闭眼 + 下垂感
│   ┌───────────────────┐    │
│   │ ② 眼球 + 瞳孔      │   │  ← 控制视线方向
│   │  ┌─────┬──────┐   │   │
│   │  │瞳   │高 光 │   │   │
│   │  │孔   │✨    │   │   │
│   │  └─────┴──────┘   │   │
│   └───────────────────┘    │
│ ③ 下眼睑 (浅粉描边)        │  ← 控制下垂程度
│ ④ 眼下竖纹腮红 (细条)      │  ← 显隐/透明度
└─────────────────────────────┘
```

**下垂关键**：外眼角必须低于内眼角 2-4px

## 八、全表情联动动作系统

### 8.2 😊 开心表情

**面部**：弯月笑眼，眼尾上扬，浅粉腮红，嘴角上扬柔和微笑，眉毛平缓

**联动动作**：

```
时间:  0    0.3    0.6    1.0    1.5    2.5    3.0
      │     │      │      │      │      │      │
歪头  ──→ 开始  → 保持   → 点头×2 → 回正  →    → idle
肩膀  ──→ 抬    → 保持   → 回落   →      →    → idle
手臂  ──→ 抬胸口→ 十指握 → 微晃   → 放下  →    → idle
蝴蝶结─→ 右飘  → 保持   → 回弹   →      →    → idle
```

| 参数 | 值 |
|------|-----|
| 歪头角度 | +6° (右) |
| 肩膀 | 抬 3px |
| 胸腔 | 扩 1.5% |
| 手部 | 胸口十指交握 |
| 蝴蝶结 | 侧飘 4px |
| 总时长 | 3.0s |

### 8.3 😢 难过表情

**面部**：下垂狗狗眼，上眼皮下压，眼尾垂落，竖纹腮红加深，眉蹙起，嘴角下撇

**联动动作**：

```
时间:  0    0.4    1.2    2.0    2.8    3.5    4.0
      │     │      │      │      │      │      │
低头  ──→ 缓慢  → 保持   → 缓慢抬 → 回正  →    → idle
含胸  ──→ 沉肩  → 保持   → 舒展   →      →    → idle
手臂  ──→ 蜷缩  → 保持   → 松开   →      →    → idle
蝴蝶结─→ 垂落  → 保持   → 回弹   →      →    → idle
```

| 参数 | 值 |
|------|-----|
| 低头 | +8° (前倾) |
| 肩膀 | 沉 4px |
| 胸腔 | 缩 1%, 呼吸变慢 50% |
| 手部 | 腹部蜷缩 |
| 蝴蝶结 | 下垂 5px |
| 总时长 | 4.0s (最慢) |

### 8.4 😮 惊讶表情

**面部**：瞳孔放大，眼睑撑开，眉大幅抬起，嘴微张圆型，腮红淡色散开

**联动动作**：

| 参数 | 值 |
|------|-----|
| 后仰 | -6° |
| 停顿 | 0.5s 定格 |
| 肩膀 | 瞬间耸 6px → 回落 |
| 胸腔 | 扩 2.5% |
| 手部 | 抬至脸颊侧，手指张开贴腮 |
| 蝴蝶结 | 后扬 8px (幅度最大) |

## 十、动作系统实现架构

### 10.1 过渡规则

```
1. 前一套动作平滑衰减 (release phase: 0.3s easeOut)
2. 中间保持最后一帧 (buffer: 0.15s)
3. 新动作缓慢加载 (attack phase: 0.25s easeIn)
4. 动作播放完毕自动回归待机
```

### 10.2 防穿模检测

| 场景 | 处理规则 |
|------|----------|
| 抬手 | 手臂图层提升 z-index |
| 手遮脸 | 手掌在头发前面 |
| 低头 | 刘海覆盖眉毛 |
| 歪头 | 蝴蝶结在发丝外侧 |

### 10.3 动作幅度安全边界

```
头部移动范围: ±12px (不超出画面)
手部移动范围: ±50px (不遮挡五官)
动作结束后: 自动回归待机姿势
```
